引入方式

Step1

在npm上下载你所需要的组件

npm install your-components --save

Step2

手动或自动link到你的项目

  • 自动方式

      react-native link  your-components

  • 手动方式

    ios

    android

Step3

在你的代码里引入并使用

import {SomeComponent} from 'your-components'
const Demo = ()=>(
    <SomeComponent />
)
